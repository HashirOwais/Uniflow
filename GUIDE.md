# User Guide

## Dashboard Overview
- The dashboard provides users with an overview of their **upcoming events** and **tasks**, along with each task’s **priority level**.  
- From the dashboard, users can **mark tasks as completed** quickly and easily.

## Calendar Access
- The **Calendar** tab at the top of the page opens the main calendar view.  
- From the calendar, users can **add new events,** and **create new groups and tasks**.

## Adding an Event
When creating an event, users can enter the following details:
- **Title**
- **Start date/time** and **End date/time**
- **Group or category** (for better organization)
- **Additional notes** (optional)

## Adding a Task
When creating a task, users can provide:
- **Title**
- **Description**
- **Deadline**
- **Priority level** — *Low*, *Medium*, or *High*

## Adding a group
When creating a group, users can make a group which acts as a bucket/category for the user to put future tasks in for enhanced organization.
